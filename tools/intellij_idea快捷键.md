### Intellij Idea快捷键

[工具激活参考](https://www.cnblogs.com/yhycoder/p/12174665.html)

**快捷键设置**：Preferences -> keymap -> Mac OS X 10.5+.

#### Mac键盘符号和修饰键说明

- `⌘` Command
- `⇧` Shift
- `⌥` Option
- `⌃` Control
- `↩︎` Return/Enter
- `⌫` Delete
- `⌦` 向前删除键（Fn+Delete）
- `↑` 上箭头
- `↓` 下箭头
- `←` 左箭头
- `→` 右箭头
- `⇞` Page Up（Fn+↑）
- `⇟` Page Down（Fn+↓）
- `Home` Fn + ←
- `End` Fn + →
- `⇥` 右制表符（Tab键）
- `⇤` 左制表符（Shift+Tab）
- `⎋` Escape (Esc)

#### Base

- `psvm`  生成main
- `sout`  生成println

- `⌘[ / ⌘]` 退回 / 前进到上一个操作的地方
